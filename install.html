Install Guide

Clean and Detailed Setup for Both Clients.

This is the full path from fresh install to a working theme, including fallback steps if your theme does not show immediately.

1

Install a Client

Choose BetterDiscord or Vencord from the Download page and complete their official installer process first.

2

Download a Theme File

Open Themes, pick your style, and use "Download Theme" to open the raw .theme.css file.

3

Save the File Correctly

Save the raw file without renaming the extension. Keep it as .theme.css and avoid adding .txt.

4

Move It to the Themes Folder

BetterDiscord: open Discord Settings > Themes > "Open Themes Folder". Place your file there. For Vencord, use its theme folder path from settings.

5

Enable and Reload

Enable the theme toggle. If changes are not visible, restart Discord once, then disable/enable the theme again.

6

Troubleshooting

If UI breaks: test another theme from this site, verify internet connection (raw URL must be reachable), and confirm no duplicate conflicting CSS files are active.
