Install Flow

Set Up Any Theme in Under 3 Minutes.

Use this sequence for BetterDiscord or Vencord.

01

Open Themes Page

Go to `Themes` and choose your preferred style.

02

Download .theme.css

Use the raw download button from the selected theme card.

03

Move into Theme Folder

Place the file in your BetterDiscord/Vencord themes directory.

04

Enable and Reload

Enable the theme in client settings and reload Discord if needed.
