Install Guide

Set Up Themes in 3 Minutes.

Download a .theme.css, move it into your themes folder, and enable it inside your client settings.

Go to Theme Downloads
Step 1: Download
Step 2: Enable

Setup Flow

Follow this sequence for each client and you should be fully set in minutes.

01

Download Theme File

Open themes.html, choose one theme, and download the raw .theme.css file.

02

Open Client Theme Folder

In BetterDiscord use "Open Themes Folder". In Vencord use Themes or QuickCSS loader.

03

Move File and Enable

Place crimson.theme.css or purple.theme.css into the folder and toggle it on.

04

Reload if Needed

If visuals do not apply instantly, reload Discord once and verify active plugins are compatible.
